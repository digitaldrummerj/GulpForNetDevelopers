Compile Project

Overview

The compile project tasks takes care of compile your project without increment the version numbers or deploying it anywhere.

  1. Run lint task
  2. Run inject task
  3. Get version number from package.json
  4. Create versioned build directory
  5. Delete existing files from the versioned build directory
  6. Compile the solution using msbuild
  7. Copy the msbuild output to the build directory.
